# Engwolf

Engwolf - это Android приложение для эффективного запоминания английских слов с помощью умного распределения повторений и разделения слов по уровням сложности

---

## Архитектура приложения

Приложение построено по трехслойной схеме
* **Data Layer** - база данных Room и JSON файлы в assets
* **Domain / Repository Layer** - WordRepository и логика интервалов
* **UI Layer** - Jetpack Compose и управление состоянием в QuizViewModel

---

## Структура базы данных

### Сущность WordEntity
Таблица words содержит следующие поля
* `id` - уникальный номер слова с автогенерацией
* `word` - слово на английском языке
* `translation` - перевод слова на русский язык
* `dictionaryName` - название словаря или уровень сложности
* `intervalStep` - текущий шаг пользователя в системе повторений
* `nextReviewTime` - время следующего показа слова в миллисекундах
* `isLearned` - отметка о том, что слово полностью выучено

### Запросы WordDao
* `getWordsToReview` - выдает список слов для повторения на основе текущего времени
* `insertWords` - добавляет пачку слов в базу данных без замены существующих
* `updateWord` - сохраняет измененное состояние конкретного слова
* `getCount` - возвращает количество слов в словаре для проверки первичной загрузки
* `getUniqueWordsStudiedToday` - считает количество уникальных слов за текущие сутки

---

## Логика интервального повторения

Повторение работает по сетке шагов со следующими временными интервалами
`новое слово -> 1 минута -> 10 минут -> 1 час -> 12 часов -> 1 сутки -> 1 неделя -> 1 месяц`

### Изменение прогресса
* **При неверном ответе** - шаг сбрасывается на 0, а время следующего показа устанавливается на начало текущего дня плюс 1 секунда, чтобы слово вернулось в пул сразу
* **При верном ответе** - шаг увеличивается на 1, но не превышает размер сетки интервалов, а новое время рассчитывается как текущее время плюс интервал шага

### Суточные лимиты
В программе установлено ограничение на изучение не более 10 новых слов в день

## Управление интерфейсом

### Экран тестирования
Логика экрана завязана на QuizViewModel, которая хранит состояние QuizUiState
* `word` - текущее проверяемое слово
* `text` - текст, введенный пользователем в поле ввода
* `feedback` - сообщение о правильности ответа
* `showNextButton` - флаг отображения кнопки переключения слова
* `isLimitReached` - флаг достижения суточного лимита слов
* `isLoading` - флаг процесса загрузки данных из базы
