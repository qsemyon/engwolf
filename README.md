# Engwolf: Изучение английского языка

Документация программы для Android, предназначенного для освоения лексического минимума уровней A1–B1. Приложение базируется на алгоритмах интервальных повторений, разработанных на основе кривой забывания Эббингауза

## Концептуальная основа
В основе системы лежит метод распределенных повторений (SRS). Программная логика рассчитывает оптимальные временные интервалы для проверки знаний, что обеспечивает перенос лексических единиц из кратковременной памяти в долгосрочную

## Технологический стек
* **Язык разработки:** Kotlin
* **UI-фреймворк:** Jetpack Compose (Material 3)
* **Архитектура:** MVVM (Model-View-ViewModel)
* **Локальное хранение данных:** Room Persistence Library (SQLite)
* **Обработка данных:** Kotlin Coroutines
* **Формат ресурсов:** JSON (статические справочники)

## Реализованный функционал

### 1. Управление данными и хранение
* **Persistence Layer:** Спроектирована схема базы данных Room для хранения прогресса обучения, временных меток и состояний интервальных шагов
* **Data Repository:** Реализован слой репозитория, обеспечивающий абстракцию над источниками данных
* **Initial Seeding:** Реализован механизм первичного наполнения базы данных путем парсинга ресурсов из JSON-файла при первом запуске приложения

### 2. Алгоритмическая часть
* **SRS Algorithm:** Интегрирована система из 7 временных интервалов (от 0 до 30 суток)
* **Progress Tracking:** Реализована логика обновления прогресса (инкремент шага при успехе, обнуление при ошибке)
* **Validation Logic:** Система автоматической валидации пользовательского ввода с нормализацией строк

### 3. Интерфейс пользователя
* **Navigation:** Настроена декларативная навигация между экранами обучения и стартовым меню
* **Core UI:** Реализованы основные компоненты: карточка активного слова, интерактивное поле ввода с обратной связью и механизм переключения состояния экранов

## План дальнейшей разработки

### 1. Модуль фонетики
* Интеграция Android Text-to-Speech API для воспроизведения эталонного произношения

### 2. Образовательный контент
* Наполнение специализированными словарями для уровней A1, A2, B1 (общим объёмом 1200 слов)
* Разработка модуля грамматики, включающего теоретические блоки и систему тестового контроля

### 3. Модуль аналитики
* Реализация графического отображения динамики обучения
* Система сбора статистики за отчётный период (7 дней)
